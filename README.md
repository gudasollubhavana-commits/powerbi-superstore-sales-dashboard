# Superstore Sales Dashboard - Power BI

## Overview

This project is an interactive Power BI dashboard built using the Superstore dataset. The dashboard provides insights into sales performance, profit trends, order volume, category-wise sales distribution, regional sales contribution, and customer segment analysis.

## Features

* Total Sales KPI
* Total Profit KPI
* Total Orders KPI
* Sales by Category Analysis
* Sales by Region Analysis
* Sales by Segment Analysis
* Interactive Filters and Visualizations

## Tools Used

* Power BI Desktop
* Data Visualization
* Business Intelligence Reporting
* Data Analysis

## Dashboard Insights

* Technology category generated the highest sales.
* West region contributed the largest share of total sales.
* Consumer segment accounted for a significant portion of overall revenue.
* Dashboard enables quick analysis of sales performance across multiple dimensions.
